## ヒアリングの目的

明日1時間のヒアリングで、以下3つを達成する。

1. **AIの可能性を腹落ちさせる**：実務者が「自分の業務で使えそう」と感じる具体度で
2. **現状の業務とデータの実態を引き出す**：データ整理状況・痛点・運用フローを把握
3. **次回提案につながる具体イメージをデモで残す**：PoC候補を一緒に選べる状態に

## 制約・前提

| 項目 | 内容 |
|---|---|
| 時間 | 60分 |
| 参加者 | 現場・実務者（設計／施工／営業） |
| 時間配分 | 教育20% / ヒアリング40% / デモ40% |
| 提供物 | HTMLスライド（投影＋PDF配布兼用） |
| デモツール | **Claude for Work（チャットUI／Claude.ai）** — チャットインターフェースで完結。ファイル添付（Excel/PDF）方式 |
| 機密情報 | デモはすべて**ダミーデータ**で実施（PSC社内データは未受領のため使わない） |
