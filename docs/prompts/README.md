# デモ用プロンプト集

明日のヒアリングデモで Claude for Work（Claude.ai）に投入するプロンプト一式。

## 使い方

1. Claude for Work にログイン
2. 各デモの対応する Excel／PDF ファイルをチャットに添付
3. 本ディレクトリの対応ファイルからプロンプトをコピペ
4. 順番通りに投入

## デモのコツ

- プロンプトは **必ずリハーサルで1回通す**。Claude の出力は揺らぐので、本番で「だいたいこうなる」を体感しておく
- 各プロンプトの末尾に「期待する出力ポイント」を記載 → リハで出ているか確認
- 出力に不要なものが混じったら、追加で「○○は省いてください」と聞き直すデモも自然
- 質問が来たら、その場で追加プロンプトを打ち込む余白を意識（時間に1分の余裕を残す）

## 失敗時の対応

- 30秒待っても応答なし → 「事前に録画したものをお見せします」と切替
- 想定と違う出力 → 「○○を中心にやり直してください」と素直に再指示
- 添付ファイルが読めない → 「Excel／PDFが読み込めないようです、こちらに貼り付けます」とテキスト貼り付けに切替
